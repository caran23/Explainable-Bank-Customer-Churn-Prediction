# Explainable and Profit-Driven Bank Customer Churn Prediction

## Overview

This project predicts whether a bank customer is likely to churn using an ensemble machine learning model. It also provides explainable AI insights to help understand the factors influencing customer churn.

The application is developed using Python and Streamlit, allowing both single customer prediction and bulk prediction through CSV file upload.

---

## Features

- Predict customer churn using an ensemble machine learning model
- Single customer prediction
- Bulk prediction using CSV upload
- Explainable AI insights
- Interactive Streamlit web application
- High prediction performance with optimized ensemble learning
